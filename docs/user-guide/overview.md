# User Guide

Learn how to create jobs, review results, configure settings, and export your work.

## Main Views

### Home
Create new recognition jobs with configurable settings for printed or handwritten text. See [Creating Jobs](creating-jobs.md) for detailed instructions.

### Job Control
Monitor and manage all jobs with status indicators and queue controls. See [Job Control](job-control.md) for details.

### Settings
Configure auto-start, notifications, concurrent jobs, working directory, and preferences. See [Settings](settings.md) for all available options.

## Workflow

1. **Create** - Select input (image/PDF/folder) and configure job. See [Creating Jobs](creating-jobs.md)
2. **Process** - Monitor progress in [Job Control](job-control.md)
3. **Review** - Correct recognition errors in side-by-side interface. See [Review Interface](review-interface.md)
4. **Export** - Save as text, DOCX, or PDF. See [Export Options](export-options.md)

## Job Types

- **Printed** - Uses Kosmos-2.5 for printed documents
- **Handwritten** - Uses TrOCR with YOLO segmentation

## Job Status

- **Pending** - Queued for processing
- **Running** - Currently processing
- **Cancelled** - Stopped manually
- **Error** - Error occurred (check logs)
- **Complete** - Ready for review
